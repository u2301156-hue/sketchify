# sketchify
An ai drawing guidance app which helps user to draw from scratch ...

Sketchify AI is a mobile app that converts user-uploaded portraits into structured step-by-step drawing tutorials using AI.

Tech Stack

Gemini API
Firebase
Pillow 

Features

Upload portrait from gallery
AI-generated 6-step drawing tutorial
Transparency overlay adjustment
Step-by-step navigation
Save generated tutorial
Dark mode support
