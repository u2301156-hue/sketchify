# sketchify
An ai drawing guidance app which helps user to draw from scratch ...

Sketchify AI is a mobile app that converts user-uploaded portraits into structured step-by-step drawing tutorials using AI.


Features

Upload portrait from gallery
AI-generated 6-step drawing tutorial
Transparency overlay adjustment
Step-by-step navigation
Save generated tutorial
Dark mode support

# 6-Step Portrait Drawing Tutorial Generator

## Project Description
This Python script generates a **6-step portrait drawing tutorial** from an uploaded image using Google's Gemini AI image model. 
Each step progressively shows the drawing process in **grayscale pencil sketch style**.

## Tech Stack & Dependencies
- Python 3.10+
- matplotlib
- Pillow (PIL)
- python-dotenv
- Google GenAI API
